# JavaScript API

The PluginDx JavaScript API exposes the following methods to the global object `PDX.panel`. You can use these methods to control the embed panel programmatically.

## .init()

```javascript
PDX.panel.init(handler);
```

## .open()

```javascript
PDX.panel.open();
```

## .close()

```javascript
PDX.panel.close();
```

## .toggle()

```javascript
PDX.panel.toggle();
```
